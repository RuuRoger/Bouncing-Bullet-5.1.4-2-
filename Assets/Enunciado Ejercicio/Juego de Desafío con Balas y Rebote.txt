Juego de Desafío con Balas y Rebotes

Descripción:
Crea un juego en el que el jugador deba disparar balas que reboten en las paredes para alcanzar objetivos. Usa Rigidbody.AddForce para disparar las balas y Collider para manejar los rebotes.

Instrucciones:
Escenario: Diseña una arena cerrada con varias paredes y obstáculos.

Personaje: Añade un objeto (como una torreta) que pueda disparar balas usando Rigidbody.

Disparo de Balas: Implementa un script para disparar balas usando Rigidbody.AddForce.

Rebotes: Configura los colliders de las balas y las paredes para que las balas reboten correctamente al chocar.

Objetivos: Añade objetos como objetivos que el jugador deba alcanzar con las balas. Usa OnTriggerEnter para detectar cuando una bala alcanza un objetivo.
